Getting Started

What is CubeKit?

CubeKit is a comprehensive collection of high-level React components designed specifically for Cube.dev applications. It provides ready-to-use components that help you build sophisticated analytics interfaces in minutes, not months.

Installation

NPM Yarn 
npm install @cubekit/react @cubekit/core
npm install @cube-dev/client-react

Quick Start

import { QueryBuilder } from '@cubekit/react';
import '@cubekit/react/dist/index.css';
// Setup Cube API
const cubeApi = cubejs('YOUR_TOKEN', {
  apiUrl: 'http://localhost:4000/cubejs-api/v1'
});

Components

QueryBuilder

A powerful no-code interface that allows business users to build complex Cube.dev queries through an intuitive drag-and-drop interface.

Live Preview

import { QueryBuilder } from '@cubekit/react';
<QueryBuilder
  cubeApi={cubeApi}
  query={query}
  onQueryChange={setQuery}
  enableFilters={true}
  theme="cubekit"
/>

Filters

Advanced filtering capabilities with multiple operators and value types.

<QueryBuilder.Filters
  filters={filters}
  onFiltersChange={setFilters}
  operators={['equals', 'contains', 'gt', 'lt']}
/>

Measures

Select and configure measures for your query with aggregation options.

<QueryBuilder.Measures
  measures={measures}
  onMeasuresChange={setMeasures}
  showAggregationOptions={true}
/>

Dimensions

Choose dimensions to group and segment your data.

<QueryBuilder.Dimensions
  dimensions={dimensions}
  onDimensionsChange={setDimensions}
  maxDimensions={5}
/>

Time Dimensions

Configure time-based analysis with granularity and date range controls.

<QueryBuilder.TimeDimensions
  timeDimensions={timeDimensions}
  onTimeDimensionsChange={setTimeDimensions}
  defaultGranularity="day"
/>

SQLBuilder

An advanced SQL query builder that works with Cube.dev's SQL API endpoint, featuring syntax highlighting, autocomplete, and validation.

Live Preview

SELECT
measure(`Orders.count`) as `orders_count`,
dimension(`Orders.status`) as `orders_status`
FROM cube(`Orders`)
WHERE dimension(`Orders.createdAt`) >= '2024-01-01'
GROUP BY dimension(`Orders.status`)
import { SQLBuilder } from '@cubekit/react';
<SQLBuilder
  cubeApi={cubeApi}
  sql={sql}
  onSqlChange={setSql}
  enableAutocomplete={true}
/>

SQL Editor

Full-featured code editor with syntax highlighting and multiple themes.

<SQLBuilder.Editor
  value={sql}
  onChange={setSql}
  theme="vs-dark"
/>

Autocomplete

Intelligent code completion for Cube.dev schema elements.

<SQLBuilder.Autocomplete
  schema={cubeSchema}
  enableSnippets={true}
/>

Validation

Real-time SQL validation against your Cube.dev schema.

<SQLBuilder.Validation
  sql={sql}
  schema={cubeSchema}
  showInlineErrors={true}
/>

AI ChatBot

Natural language query interface powered by AI that converts conversational requests into Cube.dev queries.

Live Preview

Show me total sales by month for the last 6 months
I'll create a query to show monthly sales data. Here's what I found:
SELECT SUM(amount) as total_sales, DATE_TRUNC('month', created_at) as month
FROM orders WHERE created_at >= NOW() - INTERVAL '6 months'
GROUP BY month 
import { AIChatBot } from '@cubekit/react';
<AIChatBot
  cubeApi={cubeApi}
  aiApiKey={process.env.REACT_APP_AI_API_KEY}
  onQueryGenerated={handleQueryGenerated}
  enableQuerySuggestions={true}
/>
